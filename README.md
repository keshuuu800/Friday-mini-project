# FRIDAY AI Assistant

An Iron Man-inspired AI voice assistant built with Python.

## Features
- Voice recognition
- AI conversations using Ollama
- Animated futuristic UI
- Internet search
- Text-to-speech
- Wake word detection ("Friday")

## Tech Stack
- Python
- CustomTkinter
- Ollama
- SpeechRecognition
- pyttsx3

## Run Locally

```bash
pip install -r requirements.txt
ollama serve
python friday_ui.py
